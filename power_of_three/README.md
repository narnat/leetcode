# Power of Three

Given an integer, write a function to determine if it is a power of three.

Example 1:

``` shell
Input: 27
Output: true
```

Example 2:

``` shell
Input: 0
Output: false
```

Example 3:

``` shell
Input: 9
Output: true
```

Example 4:

``` shell
Input: 45
Output: false
```

Follow up:
Could you do it without using any loop / recursion?
